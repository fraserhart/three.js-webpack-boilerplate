# three.js cardboard VR boilerplate
Test
Three.js cardboard VR boilerplate with:

* webpack
* webpack-dev-server
* three.js
* three stereo effect (https://github.com/Samsy/three-stereo-effect)
* eslint (airbnb configs)
* sasslint
* Jest

## Setup
`npm install`

## Test
`npm test`

## Dev
`npm run start`

## Build
`npm run build`
